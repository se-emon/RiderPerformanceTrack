# **App Name**: RiderTrack

## Core Features:

- Performance Metrics Display: Display key performance metrics: Success Ratio, Failed Ratio, Return Ratio, Total Riders. Metrics are color-coded (green for success, red for failed, yellow for returns).
- Daily Entry Form: Add daily entries via a form with fields for Date, Rider Name (dynamic dropdown), Successful, Failed, and Returned deliveries.
- Data Filtering: Filter entries by Start Date, End Date, and Rider, with options to reset filters, apply filters, or show all entries.
- Data Table: Display today's entries in a table with columns for Date, Rider, Delivered, Failed, Returned, Total, Success Ratio, Failed Ratio, and Returned Ratio. Display a 'No entries found' message when applicable.
- Monthly Report Generation: Generate a monthly performance report to display top performers based on success, fail, and return rates for that period.
- Entry Submission Count: Tally and visually show number of days entries were added, per month.

## Style Guidelines:

- Primary color: Deep Purple (#6750A4) to reflect reliability. This hue aims for balance between energy and trust.
- Background color: Light Lavender (#E6E0EA) provides a gentle backdrop.
- Accent color: Violet (#D0BCFF) for interactive elements, creating clear affordances.
- Font: 'Inter', a grotesque-style sans-serif, is used for both headings and body text, given its modern, neutral appearance.
- Simple, clear icons to represent metrics (success, failure, returns) and actions (add, filter, reset).
- Responsive layout that adapts to different screen sizes, ensuring usability on both desktop and mobile devices.
- Subtle transitions and animations for loading data and updating metrics to provide a smooth user experience.
